    Documentation home¶

    Getting started
    Adding your first repository
    Sign up with your Git provider so that A^O_DOCS can have access to your Git provider organizations and members. You can then add any repository you have access to with one click.
    Supported languages
    A^O_DOCS supports over 30 programming languages out of the box, and regularly adds support for new languages and tools.
    Using A^O_DOCS
    Creating and managing an organization
    A^O_DOCS automatically imports your Git provider organizations and members. Changes reflect on A^O_DOCS in real time and you can manage people who joined your organization on A^O_DOCS.
    Setting up integrations
    Set up integrations to receive A^O_DOCS status checks on your current workflow.

    Most popular topics

    Configuring code patterns
    Adapt A^O_DOCS to your scenario by configuring the code patterns used to analyze your repositories
    Adding coverage reports
    Set up your repositories to show code coverage reports directly on A^O_DOCS.
    Managing people in organizations
    Invite your team members to join your organization on A^O_DOCS so that A^O_DOCS analyzes their commits on private repositories.
